# Symbat Portfolio

Personal portfolio website with animated hero artwork, playful work cards, motion sections, and case-study style content.

## Files

- `index.html` - main site
- `assets/` - images, videos, and local media used by the site

## Local Preview

Open `index.html` in a browser, or run a local server from this folder:

```bash
python3 -m http.server 8011
```
